Introduction

This project is based on Electron and VueJS, using EaselJS for manage the canvas element. This project is an intent to make an open source powerful editor for levels.

Features

  • Manage custom tilesets
  • Level files with custom tilesets in the file so you can share with freedom
  • Export to JSON and PNG
  • Properties on Models, Intances and Level Root
  • Multiselection on models for draw randomly
  • Basic tools:
    • Selection and move
    • Brush
    • Eraser
    • Eyedrop
    • Lines
    • Rectangles
    • Ellipses
    • Bucket for fill
